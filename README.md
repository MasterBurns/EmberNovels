# 🔥 EmberNovels

EmberNovels ist eine moderne, datenschutzfreundliche und feature-reiche Desktop-App für Roman-Autoren und kreative Schreiber. Sie verbindet ablenkungsfreies Schreiben (Zen-Modus) mit mächtiger KI-Unterstützung (Lektorat, Übersetzungen) und strukturierter Weltenplanung (Lore-Wiki, Timeline).

## 🛠️ Installation & Setup / Installation

EmberNovels contains a FastAPI backend and a light vanilla JS frontend.

### Backend Setup
1. Clone the repository and navigate into it.
2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the server:
   ```bash
   python backend/main.py
   ```

### Frontend Setup
Open `frontend/index.html` in your browser, or serve it using any web server.
For updates check raw version configs on GitHub releases.
